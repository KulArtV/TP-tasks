Игра состоит из поля 150*150 и двух армий, состоящих из пехоты, танков, артиллерии и авиации. 
Также на поле будут монеты, собирая которые, можно прокачать какие-то скиллы. Армии создаются с помощью абстрактной фабрики. 
Пехота, танки, артиллерия и авиация объединены в класс Unit, и для них определены общие методы «ход» и «выстрел», таким образом для Unitов используется фабричный метод.
Также в игре допустима функция клонирования Unitа за деньги, для этого используется паттерн Prototype, 
то есть для всех подклассов класса Unit определен метод clone()

Сборку можно воспроизвести как через cmake, так и через g++ main.cpp в терминале
